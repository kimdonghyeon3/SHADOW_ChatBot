Shadow 개요

Shadow는 배움을 위해 누군가를 "따라가다" 라는 뜻입니다.

현재 디지털 소외계층은 인터넷 쇼핑을 하기에 다양한 버튼이 존재하고, 어떠한 과정을 거쳐야 하는지에 대해 큰 어려움을 겪고 있습니다. 이를 방지하기 위해 Shadow는 디지털 소외 계층에
대한 교육이나, 도움 서비스의 필요성이 높아짐에 따라 해당 계층의 편리한 온라인 서비스 기능을 활용하기 위해 인터넷 쇼핑몰의 메뉴를 찾아주고 원하는 요구 사항에 대한 해결 및 활용방법을
안내하기 위해 제작되었습니다.


Shadow가 제공하는 다양한 기능

정확한 키워드 도출

● Shadow는 Naver Clova Chatbot을 사용해 정확한 키워드를 도출해내며, 키워드를 통해 알맞은 쇼핑몰의 메뉴를 찾아줍니다.


편리한 제작

● 쇼핑몰을 제작하는 관리자가 키워드와 버튼 순서, URL, 간단한 설명을 작성하고 생성하면 쉽고 빠르고 편리하게 Shadow를 적용할 수 있습니다.


이용자의 편리한 사용

● 관리자가 적용한 Shadow를 사용해 사용자는 자유롭게 원하는 질문을 작성하고, 원하는 목적지까지 올바르게 도달할 수 있도록 도와줍니다.
