Introduction


About Shadow Service

Shadow는 인터넷 쇼핑몰의 아이콘을 설명하고, 고객이 이용하고자 하는 메뉴를 찾아주며, 원하는 요구사항에 대한 방법을 안내하는 서비스입니다.


Shadow Page

Shadow 메인페이지 입니다.


Make New Shadow


1. Login

강조되어 있는 로그인 버튼을 클릭하여 로그인 화면으로 이동합니다.


2. Login Form

아이디와 비밀번호를 입력 후 로그인 버튼을 클릭합니다.


3. Make Shadow

새로운 Shadow를 만들기 위해 강조되어 있는 Make Shadow버튼을 클릭합니다.


4. Insert Info

새로만들 Shadow의 정보를 입력 후 챗봇 생성 버튼을 클릭합니다.


4-1. Add/remove Keyword

키워드 추가 혹은 삭제를 위해 버튼을 클릭합니다.


5. Created successfully!

Smart한 쇼핑을 위한 Shadow가 성공적으로 생성되었습니다!



Shadow Management


1. Shadow List

Shadow에 저장되어 있는 쇼핑몰의 정보를 수정합니다.


1-1. Shadow Modify

1번 버튼을 클릭하여 만들어 둔 Shadow List를 확인합니다. 그리고 2번 버튼을 클릭하여 수정을 시작합니다.


1-2. Shadow Modify2

Shadow에 저장되어 있는 쇼핑몰의 정보를 수정합니다.


2-1. Shadow Delete

1번 버튼을 클릭하여 만들어 둔 Shadow List를 확인합니다. 그리고 2번 버튼을 클릭하여 삭제를 시작합니다.


2-2. Shadow Delete2

삭제 버튼 클릭 시 띄워지는 창의 예 버튼을 클릭합니다.


2-3. Shadow Delete3

"예" 버튼을 클릭하면 성공적으로 Shadow가 삭제됩니다.


Apply shadow


1. Shadow List Page

강조되어 있는 List 버튼을 클릭하여 List 화면으로 이동하고, 쇼핑몰 이름을 클릭하여 Detail 페이지로 이동합니다.


2. Code

강조되어 있는 코드를 복사합니다.(복사버튼을 사용하시면 편리합니다.)

만일 해킹의 위협이 있을 경우 갱신버튼을 사용하시면 API KEY가 갱신됩니다.


3. Apply Your Code

HTML head 부분에 코드를 넣어주세요. 당신만의 Shadow가 활성화됩니다!
